# 欢迎来到我的博客 🎉

这是我的个人博客，使用 **Next.js** 全栈框架构建，界面设计借鉴了 [Firefly](https://github.com/CuteLeaf/Firefly) 的 Material Design 3 风格。

## 技术栈

- **Next.js 16** - React 全栈框架
- **TypeScript** - 类型安全
- **Tailwind CSS v4** - 样式框架
- **MDX** - Markdown 文章渲染

## 功能特点

### ✨ Material Design 3 风格
柔和、现代的 Material Design 3 设计，支持亮色/暗色模式切换。

### 📝 Markdown 文章
使用 Markdown/MDX 编写文章，支持代码高亮、表格、GFM 语法。

### 🏷️ 标签和分类
文章支持标签和分类管理，方便浏览和检索。

### 🌙 深色模式
自动跟随系统偏好，也可手动切换。

## 示例代码

```typescript
function greet(name: string): string {
  return `Hello, ${name}! Welcome to my blog.`
}

console.log(greet('访客'))
```
